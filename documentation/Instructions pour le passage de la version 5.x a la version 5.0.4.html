Consignes de mise à jour du script AlexGuestbook
de la version 5.0.* à la version 5.0.4


Cette mise à jour du livre d'or AlexGuestbook est vivement recommandée. Elle corrige plusieurs failles de sécurité, supporte PHP > v5.2 et corrige quelques bugs.
La liste des modifications et corrections apportées au script est disponible dans le fichier changelog_fr.txt.
Retrouvez toutes les informations, les mises à jour et l'assistance du script sur www.alexguestbook.net


Avant toutes modifications, faites une sauvegarde de la base de données (voir l'interface d'administration du script) et de vos skins si vous les avez personnalisés.


  1. Depuis la version 5.0.3, AlexGuestbook propose une nouvelle façon de mettre à jour le script AlexGuestbook. Un script de mise à jour est à télécharger sur notre site www.alexguestbook.net. Téléchargez également l'archive complète du script AlexGuestbook 5.0.4.

  2. Vous devez extraire tous les fichiers du script version 5.0.4 et les envoyer (avec votre logiciel FTP, par exemple FileZilla) sur votre serveur dans le répertoire du livre d'or (par défaut "agb"). Ecrasez tous les fichiers sauf les skins si vous les avez modifiés.

  3. Faites de même avec le fichier agb-update.zip : décompressez le fichier ZIP et envoyez le répertoire "update" sur votre serveur toujours dans le répertoire du livre d'or.

  4. Lancez la mise à jour en vous connectant depuis votre navigateur (Internet Explorer, Firefox, Safari ou autre) à l'adresse www.votresite.fr/repertoire_du_livre_dor/update/index.php (il est nécessaire d'ajouter le /update/index.php pour accéder à l'interface de mise à jour).

  5. Suivez les instructions qui vous sont données dans l'interface de mise à jour.

  6. Si vous avez modifié les skins, vous devez les mettre à jour (nouveau copyright). La procédure de mise à jour du skin est expliquée dans l'interface de mise à jour.

  7. Si vous utilisez l'option "include", vous devez modifier la variable $chem_absolu dans le fichier index.php du livre. La procédure de mise à jour du skin est expliquée dans l'interface d'administration du livre ou sur notre site www.alexguestbook.net rubrique FAQ.

Bonne mise à jour à tous !
