1) Installation
 
  • Envoyez tout le répertoire agb (renommez-le si vous le désirez, par exemple "livredor") sur le serveur hébergeant votre site avec votre logiciel FTP (par exemple FileZilla).
  • L'installation est automatique, vous devrez juste donner quelques informations et suivre les instructions : ouvrez votre navigateur (Internet Explorer, Firefox...) et allez sur la page "setup.php" du répertoire principal (par exemple : http://www.mon_site.com/agb/setup.php).
  • Votre script est installé

- La partie administration se trouve dans le répertoire "admin" du script, url du style :
  http://www.mon_site.com/agb/admin/

- La partie affichage se trouve à la racine du répertoire du script, url du style :
  http://www.mon_site.com/agb/
