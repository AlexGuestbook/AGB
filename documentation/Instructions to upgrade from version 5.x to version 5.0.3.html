Instructions for updating the AlexGuestbook script
from version 5.0.* to version 5.0.3


This update of AlexGuestbook is highly recommended. It fixes several security vulnerabilities, supports PHP > v5.2 and fixes some bugs.
The list of modifications and corrections to the script is available here : changelog_en.txt.
Find all the information, updates and help for the script on www.alexguestbook.net


Before making any changes, make a backup of the database (see the admin interface of the script) and of your skins if you have customized them.


  1. Version 5.0.3 offers a new way to update the AlexGuestbook script. An updating script is available for download on our website www.alexguestbook.net. Also download the complete archive of the AlexGuestbook 5.0.3 script.

  2. You must unzip all files of the script version 5.0.3 and send them (with your FTP program, FileZilla for example) on your server in the guestbook directory (by default "agb"). Overwrite all files, except the skins if you have modified them.

  3. Do the same with the update-5.0.3.zip file : unzip it and send the folder "update" on your server, still in the guestbook directory.

  4. Start the update by logging in with your browser (Internet Explorer, Firefox, Safari or other) at www.yoursite.com/guestbook_directory/update/index.php (it is necessary to add /update/index.php to access the update interface).

  5. Follow the instructions given in the update interface.

  6. If you have modified the skins, you must update them (new copyright). The procedure for updating the skin is explained in the update interface.

  7. If you use option "include", you must modify variable $chem_absolu in file index.php of guestbook. The procedure for updating the skin is explained in the administration panel or on www.alexguestbook.net section FAQ.

Good update to all!
