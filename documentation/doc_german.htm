1) Installation
 
  • Mit beliebiger FTP Client-Software (z.B. Filezilla), das ganze agb Verzeichnis (je nach Wunsch umbenennen, z.B. "guestbook") auf den Server der Ihre Website hostet hinaufladen.
  • Die Installation danach läuft automatisch ab, man muss nur einige Felder ausfüllen bzw. den Anweisungen am Bildschirm folgen. Um die Installation zu starten einfach beliebigen Web-Browser öffnen (Firefox, Safari, Konqueror, Internet Explorer, ...) und die Seite "setup.php" die sich im Hauptverzeichnis befindet (z.B. http://www.my_site.com/agb/setup.php) aurufen.
  • Ihr Gästebuch ist nun installiert

  • Die Administrationsoberfläche ist im Verzeichnis "admin" z.B.:
    http://www.my_site.com/agb/admin/
  • Das öffentlich sichtbare Gästebuch ist im Stammverzeichnis der Installation z.B.:
    http://www.my_site.com/agb/
